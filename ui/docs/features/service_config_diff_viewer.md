# Service Config Diff Viewer

The **Service Config Diff Viewer** provides a visual comparison of configuration changes before they are applied. This feature enhances safety and transparency when updating service configurations.

## Features

- **Visual Diff**: Side-by-side comparison of the original and modified configuration.
- **YAML Formatting**: Configuration is presented in YAML format for readability.
- **Change Highlighting**: Additions, deletions, and modifications are clearly highlighted.
- **Review Workflow**: Users are prompted to review changes before confirming the update.

## Usage

1. Navigate to the Service Detail page of any service.
2. Click the **Edit Config** button.
3. Make changes to the configuration (e.g., update the name, address, or parameters).
4. Click **Review Changes**.
5. The Diff Viewer will appear, showing the differences between the current configuration and the proposed changes.
6. Click **Confirm & Save** to apply the changes, or **Back to Edit** to continue modifying.

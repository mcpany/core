# Resource Preview Modal

The Resource Preview Modal allows users to view resource content in a large, dedicated dialog box, providing a better viewing experience for large files or complex data structures compared to the split-pane view.

![Resource Preview Modal](../../.audit/ui/2026-01-15/resource_preview_modal.png)

## Features

- **Large Viewing Area**: Maximizes screen real estate for content viewing.
- **Syntax Highlighting**: Automatically detects and highlights JSON, YAML, XML, Markdown, and other text formats.
- **Copy & Download**: Quickly copy content to clipboard or download as a file.
- **Context Menu Integration**: Open the modal directly from the resource list via the right-click menu.

## Usage

1.  Navigate to the **Resources** tab.
2.  Select a resource from the list.
3.  Click the **Expand** button in the preview pane header, or right-click the resource in the list and select **Preview in Modal**.

# Trace Inspector

The **Trace Inspector** provides a powerful interface for debugging MCP tool calls and upstream service interactions.

## Features

- **Advanced Filtering**: Filter traces by Status (Success/Error) and Method.
- **Detailed Inspection**:
    - **Headers**: View full request and response headers.
    - **Payloads**: Inspect formatted JSON request and response bodies with syntax highlighting.
    - **Waterfall**: Visualize the latency of the request.
- **Replay**: Quickly replay any tool call in the Playground with the same arguments.
- **cURL Export**: One-click copy of the request as a cURL command for external testing.

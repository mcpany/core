# Prompts Library

**Status:** Implemented

## Goal
Discover and use pre-defined prompt templates. Servers can expose standardized prompts (e.g., "Analyze Code", "Summarize Text") to be used by clients or the Playground.

## Usage Guide

### 1. Browse Prompts
Navigate to `/prompts`.
- **List**: Shows prompt names and descriptions.
- **Arguments**: detailed list of required input parameters.

![Prompts List](screenshots/prompts_list.png)

### 2. Use Prompt
Select any prompt from the library to view details, configure arguments, and click **"Generate Preview"** to test execution. You can then copy the output or click **"Open in Playground"**.

# Mobile Experience

**Status:** Implemented

## Goal
Manage the MCP ecosystem from any device. The UI is fully responsive, adapting layouts for tablets and mobile phones.

## Usage Guide

### 1. Adaptive Layouts
On smaller screens:
- **Sidebar**: Collapses into a hamburger menu.
- **Tables**: Optimize display by hiding non-essential columns.
- **Graphs**: Network graph simplifies touch interactions.

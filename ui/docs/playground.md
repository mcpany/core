# Interactive Playground 2.0

The Interactive Playground allows developers to test MCP tools directly from the web interface.

## Features

-   **Auto-generated Forms**: Automatically generates forms based on the tool's JSON Schema.
-   **Complex Schema Support**: Supports nested objects, arrays, and all primitive types.
-   **Validation**: Enforces required fields and type constraints.
-   **Execution History**: Shows a chat-like history of tool executions and results.

## Usage

1.  Navigate to the **Playground** page.
2.  Click **Available Tools** to see the list of registered tools.
3.  Click **Use Tool** on any tool.
4.  Fill out the form.
    -   For **Arrays**, use the "Add Item" button.
    -   For **Objects**, fill in the nested fields.
5.  Click **Run Tool**.
6.  View the JSON request and the result in the chat window.

# Mobile View Optimization

## Overview
This feature brings "Apple-style" mobile optimization to the MCP Any dashboard. It ensures that critical management tools like the Network Graph, Log Stream, and Secrets Manager are fully functional and aesthetically pleasing on smaller screens.

## Features
- **Responsive Network Graph**:
    - Control card collapses on mobile to save screen space.
    - Sheet (drawer) width adjusts to full screen on mobile.
    - Simplified layout for graph interactions.
- **Mobile Log Stream**:
    - Stacked header controls for better accessibility.
    - Optimized row layout (hides non-critical columns on mobile).
    - Mobile-friendly Pause/Resume controls.
- **Responsive Secrets Manager**:
    - Horizontally scrollable tables for small screens.
    - Stacked form inputs in the "Add Secret" dialog.
    - Improved readability of secret names and providers.
- **Global Search**:
    - Search bar adapts to mobile width.

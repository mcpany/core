# **App Name**: MCPAny Manager

## Core Features:

- Server Connection: Connect to an MCPAny server using its address and port.
- Real-time Log Display: Display real-time server logs and console output.
- Command Execution: Send commands to the MCPAny server.
- Server Status: Display basic server status information, like version and online status.

## Style Guidelines:

- Dark background (#121212) for a modern server management interface.
- Primary color: Green (#98FF98) for highlighting important server information and active elements.
- Accent color: White (#FFFFFF) for text and key UI elements, ensuring high contrast and readability.
- Body and headline font: 'Inter', a grotesque-style sans-serif, providing a modern and objective feel for both headlines and body text.
- Simple, monochrome icons to represent server status, actions, and information.
- Clean and organized layout with clear sections for logs, console input, and server status.
- Subtle animations for server connection status changes and log updates.

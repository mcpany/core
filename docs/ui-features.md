# MCP Any UI Features

This document outlines the new features implemented in the MCP Any UI Overhaul.

## Dashboard
The dashboard provides real-time metrics and service health status.

![Dashboard](../../.audit/ui/2025-12-25/dashboard.png)

## Services Management
Manage connected upstream services with toggle functionality.

![Services](../../.audit/ui/2025-12-25/services.png)

## Tools Management
View, search, and toggle available tools.

![Tools](../../.audit/ui/2025-12-25/tools.png)

## Resources Management
Manage static resources like API specs and logs.

![Resources](../../.audit/ui/2025-12-25/resources.png)

## Prompts Management
Manage and toggle prompt templates.

![Prompts](../../.audit/ui/2025-12-25/prompts.png)

Configure execution profiles (Dev/Prod), Logging, and Audit settings.

![Profiles](../../.audit/ui/2025-12-25/settings_profiles.png)

## Webhooks
Configure and test event webhooks.

![Webhooks](../../.audit/ui/2025-12-25/settings_webhooks.png)

## Middleware Pipeline
Visual management of the middleware request processing pipeline.

![Middleware](../../.audit/ui/2025-12-25/settings_middleware.png)

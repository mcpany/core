# Feature Inventory

## Active & High Priority
- **Policy Firewall Engine:** Rego/CEL based hooking for tool calls.
- **HITL Middleware:** Suspension protocol for human-in-the-loop approvals.
- **Recursive Context Protocol (RCP):** Context inheritance for subagents.
- **Shared Blackboard KV Store:** Transactional memory for agent swarms.
- **Granular Scopes:** Capability-based token system.

## Proposed: 2026-02-24
- **OpenClaw Secure Bridge:** A pre-configured adapter that wraps OpenClaw agents, enforcing Zero Trust policies.
- **Dynamic Tool Search API:** A high-performance discovery endpoint optimized for large-scale toolsets (inspired by Claude Code).
- **Agent Skill Export (Virtual Mounts):** Directory-based tool projection for legacy agents.

## Deprioritized
- **Preset Service Gallery:** Moving to "Community-Driven Marketplace" (Marketplace sub-project).
- **Windows Filesystem Locking Fix:** Low impact compared to Security/Orchestration priorities.

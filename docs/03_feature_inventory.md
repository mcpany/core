# Feature Inventory: MCP Any

## Strategic Priority (P0)
- **Policy Firewall Engine:** Rego/CEL based hooking for tool calls.
- **HITL Middleware:** User approval flows for sensitive actions.
- **Isolated Named Pipe Adapter (NEW):** Secure inter-agent communication replacing local HTTP.
- **Recursive Scope Enforcer (NEW):** Capability-based access control for nested agent calls.

## Core Capabilities (P1)
- **Recursive Context Protocol:** Standardized headers for context inheritance.
- **Shared Key-Value Store:** "Blackboard" tool for agent memory.
- **Smart Tool Aggregator (NEW):** Deduplication and prioritization to mitigate "Discovery Fatigue".
- **Tool Playground & Explorer:** Auto-generated forms and result visualization.

## Enhanced Observability (P2)
- **Live Marble Diagrams:** Visualization of concurrent agent flows.
- **Agent Black Box Player:** Replay of recorded agent sessions.
- **Cost & Metrics Dashboard:** Token usage and performance tracking.

## Maintenance & DevX (P3)
- **Interactive Setup Wizard:** Guided configuration.
- **Secret Rotation Helper:** Security maintenance tool.
- **Config Schema Migration:** Automated upgrades.

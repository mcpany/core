# Feature Inventory: MCP Any

## High Priority (Q1 2026)
*   **Zero Trust Inter-Agent Bus** (New): Secure, isolated communication channels for agent swarms.
*   **Standardized Context Inheritance** (New): Policy-driven state sharing across agent profiles.
*   **OAuth 2.1 Integration** (From MCP Spec): Align with June 2025 MCP spec for Resource Server classification.
*   **Structured Tool Output Support**: Implementation of the new MCP capability for complex data returns.

## Medium Priority (Q2 2026)
*   **Proactive Execution Guardrails**: Budgeting and policy enforcement for cron-based agents.
*   **Server-Initiated User Elicitation**: Implementation of the MCP elicitation capability.
*   **Resource Link Enrichment**: Deep linking resources within tool results.

## Proposed/Exploratory
*   **Multi-Agent "Council" Templates**: Pre-configured swarm architectures for common tasks.
*   **Zero-Knowledge Context Proofs**: Allow agents to verify context without seeing sensitive raw data.

## Strategic Priority Shift [2026-02-16]
*   **Deprioritize**: JSON-RPC Batching (Removed from MCP Spec).
*   **Elevate**: Inter-agent security and isolation to counter "Yolo Mode" adoption.

# Feature Inventory: MCP Any

## High Priority (P0)
- **Policy Firewall Engine:** Rego/CEL based hooking for tool calls to enforce security.
- **HITL Middleware:** Human-in-the-loop approval flows for sensitive tool executions.
- **Recursive Context Protocol:** Standardized headers for context inheritance in subagents.

## Medium Priority (P1)
- **Shared Key-Value Store:** "Blackboard" tool for inter-agent state coordination.
- **Granular Scopes:** Capability-based token system for fine-grained access control.
- **Tool Execution Timeline:** Visual waterfall chart for debugging latency.

## Low Priority (P2)
- **Plugin Marketplace:** Discover and install community MCP servers.
- **Cost & Metrics Dashboard:** Token usage and performance visualization.

## Proposed Additions: [2026-02-22]
- **Zero Trust Named Pipes:** Isolated inter-agent communication for local execution.
- **Context-Aware Tool Discovery:** Dynamic tool pruning based on subagent roles.

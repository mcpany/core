# Stack Configuration Editor

The Stack Configuration Editor provides a direct interface for managing your MCP Any stack configurations (YAML/JSON).

## Features

- **Web-Based Editor:** Edit your `config.yaml` directly in the browser with line numbers and monospace font.
- **Validation:** Real-time YAML syntax validation with visual feedback.
- **Stack Management:** Reset changes, export configuration to file, and save changes to the backend.
- **Enterprise UI:** Polished interface with status indicators and "Unifi"-style design.
